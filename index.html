DiffLens

A clean, secure, and privacy-focused git diff viewer. Process and share diffs with encrypted URLs - everything happens client-side with no data leaving your browser.

Try Diff Viewer View on GitHub

Why Choose DiffLens?

Built with privacy and security in mind, DiffLens processes all your code diffs locally in your browser.

Privacy First

Your code never leaves your browser. Everything is processed locally with zero data transmission.

Fast & Reliable

Instant diff processing with syntax highlighting and multiple view modes for optimal readability.

Secure Sharing

Share diffs with encrypted URLs. Content is encoded directly in the link for maximum privacy.

Open Source

Completely open source and transparent. Audit the code, contribute, or fork for your needs.

How It Works

Simple, secure, and straightforward. Get beautiful diff visualizations in just a few steps.

1

Paste or Upload

Copy your git diff output or upload a diff file directly to the viewer.

2

Choose View Mode

Select between unified or side-by-side diff views for optimal readability.

3

Analyze Changes

Review your changes with syntax highlighting and clear visual indicators.

4

Share Securely

Generate shareable links with encrypted URLs, copy to clipboard, or download the formatted diff.
